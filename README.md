# Employee Workspace Portal

## Folder Name
`PRACTICE_ASSIGNMENT1`

## Project Name
`Employee Workspace Portal`

## Small Description
Employee Workspace Portal is a simple PHP, HTML, CSS, and JavaScript based web project. It shows employee profiles, workspace details, and salary-related information in a clean dashboard layout. The project is designed to run locally in XAMPP.

## Files and Folders
- `index.php` - main entry file that loads employee data and renders the portal
- `css/style.css` - stylesheet for the full UI
- `js/script.js` - JavaScript file for search and panel interactions
- `images/` - contains screenshots and image assets used in the project
- `README.md` - project documentation

## How to Implement, Store, and Run Using XAMPP
1. Install and open XAMPP.
2. Copy the project folder `PRACTICE_ASSIGNMENT1` into the `htdocs` directory.
3. Start the Apache server from the XAMPP Control Panel.
4. Open your browser and visit this URL:
   `http://localhost/internship2026/Practice_Assignment1/`
5. The portal will load from `index.php` and display the employee dashboard.
